# 边框素材准备说明

## 📋 素材要求

### 1. 尺寸规格
- **尺寸**：240×240px
- **格式**：PNG（支持透明背景）
- **背景**：透明底（alpha通道）

### 2. 分类目录结构

需要在项目中创建以下目录结构：

```
/images/frames/
  ├── gongxifacai/     # 恭喜发财分类
  │   ├── frame1.png
  │   ├── frame2.png
  │   └── frame3.png
  ├── jixiangruyi/     # 吉祥如意分类
  │   ├── frame1.png
  │   ├── frame2.png
  │   └── frame3.png
  └── tongyongzhufu/   # 通用祝福分类
      ├── frame1.png
      ├── frame2.png
      └── frame3.png
```

## 🎨 素材设计建议

### 1. 边框设计原则
- 边框应该围绕在图片四周
- 中间区域应该透明或半透明，让用户照片显示出来
- 边框装饰应该符合分类主题

### 2. 示例设计
- **恭喜发财**：金色边框、元宝装饰、红色元素
- **吉祥如意**：传统纹样、祥云图案、吉祥元素
- **通用祝福**：简洁边框、通用装饰、百搭风格

## 📝 代码配置

### 1. 修改边框数据

在 `pages/frame-select/frame-select.js` 中修改 `loadFrames` 方法：

```javascript
loadFrames(category) {
  const frameData = {
    gongxifacai: [
      { 
        id: 1, 
        name: '边框1', 
        preview: '/images/frames/gongxifacai/frame1.png',
        frame: '/images/frames/gongxifacai/frame1.png'  // 实际边框图片
      },
      { 
        id: 2, 
        name: '边框2', 
        preview: '/images/frames/gongxifacai/frame2.png',
        frame: '/images/frames/gongxifacai/frame2.png'
      },
      // ... 更多边框
    ],
    // ... 其他分类
  }
  
  this.setData({ frames: frameData[category] || [] })
}
```

### 2. 使用云存储（推荐）

如果边框图片较多，建议上传到云存储：

```javascript
// 在云存储中创建目录结构
// frames/gongxifacai/frame1.png
// frames/jixiangruyi/frame1.png
// frames/tongyongzhufu/frame1.png

// 然后在代码中使用云存储路径
const frameData = {
  gongxifacai: [
    { 
      id: 1, 
      name: '边框1', 
      preview: 'cloud://your-env-id.frames/gongxifacai/frame1.png',
      frame: 'cloud://your-env-id.frames/gongxifacai/frame1.png'
    }
  ]
}
```

## 🚀 快速开始

### 方法1：使用本地图片
1. 在项目根目录创建 `/images/frames/` 目录
2. 按照分类创建子目录
3. 将边框PNG图片放入对应目录
4. 修改 `frame-select.js` 中的路径

### 方法2：使用云存储
1. 在微信开发者工具中打开云开发控制台
2. 进入"存储"页面
3. 创建 `frames` 文件夹
4. 按照分类上传边框图片
5. 修改代码使用云存储路径

## 📌 注意事项

1. **图片大小**：建议每个边框图片小于100KB，确保加载速度
2. **透明背景**：必须使用PNG格式，支持透明背景
3. **尺寸一致**：所有边框必须是240×240px
4. **命名规范**：建议使用有意义的文件名，如 `gongxifacai_gold.png`

2. 测试图片上传和合成功能
3. 确认透明背景正确显示
