# 小程序测试指南

## 快速测试步骤

### 1. 检查小程序是否正常启动
- 在微信开发者工具中打开项目
- 确保编译成功，没有错误
- 首页应该显示两个功能卡片："视频上传" 和 "图片处理"

### 2. 测试视频上传功能
1. 点击首页的"视频上传"卡片
2. 点击"点击上传视频"按钮
3. 选择或拍摄一个视频（建议10秒内）
4. 视频上传后应该显示预览
5. 点击"进入编辑"按钮跳转到编辑页面

### 3. 测试视频编辑功能
- **剪辑功能**：拖动滑块调整开始和结束时间
- **裁剪功能**：调整X、Y、宽度、高度参数
- **添加文字**：点击"+ 添加文字"按钮，然后点击"编辑"修改文字内容
- **预览模式**：点击"预览模式"按钮查看微信聊天界面效果
- **导出GIF**：点击导出按钮（需要云函数支持）

### 4. 测试图片处理功能
1. 点击首页的"图片处理"卡片
2. 分别上传横幅、头像、图标
3. 点击"一键导出三张PNG图片"

## 常见问题排查

### 问题1：页面显示空白
**解决方案**：
- 检查 `app.json` 中的页面路径是否正确
- 确保所有页面文件都存在（.js, .wxml, .wxss, .json）
- 查看控制台是否有错误信息

### 问题2：点击按钮没有反应
**解决方案**：
- 检查按钮的 `bindtap` 事件是否绑定正确
- 查看控制台是否有JavaScript错误
- 确保对应的方法在 .js 文件中已定义

### 问题3：视频无法播放
**解决方案**：
- 确保视频格式支持（mp4等）
- 检查视频路径是否正确
- 尝试使用真机调试（模拟器可能不支持某些视频格式）

### 问题4：样式显示不正常
**解决方案**：
- 检查 .wxss 文件是否存在
- 确保样式类名拼写正确
- 清除缓存后重新编译

### 问题5：GIF导出不工作
**说明**：GIF导出功能需要配置云函数才能完整使用。当前版本会显示提示信息。

## 调试技巧

1. **查看控制台**：在开发者工具中查看 Console 面板的错误信息
2. **使用真机调试**：某些功能在模拟器中可能不工作，需要在真机上测试
3. **检查网络请求**：如果使用云函数，检查网络请求是否成功
4. **逐步测试**：先测试基本功能，再测试复杂功能
