# 部署云函数 - 终极方法

## 🎯 最简单的方法：在微信开发者工具的文件管理器中操作

### 步骤1：找到文件管理器
1. 在微信开发者工具中，**不要**在云开发控制台操作
2. 在**左侧文件列表**中（就是显示代码文件的地方）
3. 找到 `cloud` 文件夹
4. 展开，找到 `videoToFrames` 文件夹

### 步骤2：右键点击文件夹
1. **在微信开发者工具的文件管理器中**（不是云开发控制台）
2. 右键点击 `videoToFrames` 文件夹
3. 应该会看到一个菜单

### 步骤3：查找部署选项
在右键菜单中，查找以下选项之一：
- "上传并部署：云端安装依赖" ⭐ **优先选择这个**
- "上传并部署：所有文件"
- "上传并部署：仅上传云端"
- "部署"
- "上传"

**如果还是没有这些选项，继续看下面的方法**

## 🔧 方法二：通过云开发控制台直接上传

### 步骤1：打开云函数详情
1. 在云开发控制台的云函数列表中
2. 点击 `videoToFrames` 这一行（不是右键，是左键点击）
3. 或者点击"版本与配置"

### 步骤2：查找上传按钮
在打开的页面中，查找：
- "上传代码"
- "上传并部署"
- "重新部署"
- "更新代码"

### 步骤3：选择文件
1. 点击上传按钮后
2. 选择 `cloud/videoToFrames` 文件夹
3. 或者选择 `cloud/videoToFrames/index.js` 文件

## 🆘 如果以上都不行：手动复制代码

### 步骤1：打开云函数代码编辑器
1. 在云开发控制台
2. 点击 `videoToFrames` 的"版本与配置"
3. 查找"在线编辑"或"编辑代码"按钮
4. 点击进入代码编辑器

### 步骤2：复制本地代码
1. 在微信开发者工具中
2. 打开 `cloud/videoToFrames/index.js` 文件
3. 全选代码（Cmd+A 或 Ctrl+A）
4. 复制（Cmd+C 或 Ctrl+C）

### 步骤3：粘贴到云函数编辑器
1. 在云开发控制台的代码编辑器中
2. 全选旧代码并删除
3. 粘贴新代码（Cmd+V 或 Ctrl+V）
4. 点击"保存"或"部署"按钮

## ✅ 确认部署成功

部署后：
1. 查看云函数列表中的"最后更新时间"是否更新
2. 测试上传视频，查看控制台日志
3. 应该能看到云函数被调用的日志
