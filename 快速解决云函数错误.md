# 快速解决云函数错误

## 问题说明

如果你看到"处理失败"或"云函数未配置"的错误，说明云函数还没有部署。

## 最简单的解决方法（3步）

### 第1步：打开云开发控制台

1. 在微信开发者工具中，点击顶部菜单栏的 **"云开发"** 按钮
2. 如果提示登录，请扫码登录

### 第2步：创建云函数

1. 在云开发控制台中，点击 **"云函数"** 标签
2. 点击 **"新建云函数"** 按钮
3. 输入函数名称：`videoToFrames`
4. 选择运行环境：**Node.js**
5. 点击 **"确定"**

### 第3步：上传云函数代码

1. 在项目左侧文件列表中，找到 `cloud/videoToFrames` 文件夹
2. **右键点击**这个文件夹
3. 选择 **"上传并部署：云端安装依赖"**
4. 等待上传完成（可能需要1-2分钟）

## 需要创建两个云函数

你需要创建两个云函数：

1. **videoToFrames** - 用于将视频转换为序列帧
2. **framesToGif** - 用于将序列帧合成为GIF

对第二个云函数重复上面的步骤，只是函数名称改为 `framesToGif`。

## 如果还是不行

如果按照以上步骤操作后还是报错，请检查：

1. **云开发是否已开通**
   - 在小程序管理后台（https://mp.weixin.qq.com）检查云开发是否已开通

2. **环境ID是否配置**
   - 打开 `app.js` 文件
   - 找到 `wx.cloud.init` 这一行
   - 确保配置了环境ID（如果没有，参考"云开发配置详细步骤.md"）

3. **云函数是否上传成功**
   - 在云开发控制台的"云函数"页面
   - 应该能看到你创建的云函数
   - 状态应该显示为"正常"

## 重要提示

⚠️ **云函数需要配置FFmpeg才能实际工作**

目前创建的云函数只是框架代码，要真正处理视频，还需要：
- 配置FFmpeg环境（比较复杂）
- 或者使用第三方视频处理服务

如果你只是想测试界面功能，可以先跳过云函数配置，但实际处理功能需要云函数支持。
